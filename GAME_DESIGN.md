# GAME_DESIGN.md — Urban Stack 기획서

## 게임 컨셉
도시 개발자가 되어 빌딩 블록을 부지(그리드)에 배치하며 도시를 완성하는 퍼즐 게임.
랜덤으로 공개되는 블록을 최적의 위치에 쌓아 점수를 최대화한다.

## 원작 참고
- **FITS** 보드게임의 메커니즘을 기반으로 함 (그리드 + 카드 덱 + 라운드 점수판)
- Urban Stack 고유 변형: **블록 회전·뒤집기 허용** (FITS 원작은 고정 형태)
- 테마, 비주얼, 네이밍은 완전 오리지널 (저작권 문제 없음)
- 규칙 자체는 저작권 보호 대상이 아님

## 핵심 메커니즘

### 그리드
- 크기: **5열 × 9행**
- 블록은 위(행 0)에서 아래(행 8) 방향으로 슬라이딩하여 배치
- 블록은 중력처럼 최대한 아래로 내려가서 고정됨

### 블록 (테트로미노 7종)
```
I : □□□□
O : □□
    □□
T : □□□
     □
S :  □□
    □□
Z : □□
     □□
L : □
    □
    □□
J :  □
     □
    □□
```

### 블록 변형 (Urban Stack 룰)
원작 FITS는 카드에 그려진 형태 그대로만 배치하지만, Urban Stack은 **회전과 뒤집기를 허용**한다 (캐주얼 난이도 조절).

- **회전:** 90° 시계방향. 회전 횟수 제한 없음
- **뒤집기:** 좌우 미러. 횟수 제한 없음
- 회전·뒤집기는 배치 전에만 가능 — 일단 배치되면 그리드에 고정

대칭성으로 인한 변환 결과:
- O: 회전·뒤집기 무효 (변화 없음)
- I: 회전 시 가로↔세로
- T: 4가지 방향, 뒤집기는 자기 자신
- S ↔ Z: 뒤집기로 서로 변환됨
- L ↔ J: 뒤집기로 서로 변환됨

### 카드 덱
- 7종 테트로미노를 각 N장씩 섞어 덱 구성
- 한 번에 1장씩 공개
- 공개된 블록을 회전/뒤집기로 원하는 방향 조정 후 5개 열 중 하나에 배치

### 조작
| 키 | 동작 |
|---|---|
| ← → | 열 이동 |
| ↑ 또는 X | 90° 시계방향 회전 |
| F | 좌우 뒤집기 |
| Space | 중력 낙하 배치 |
| R | 게임 다시 시작 |

### 라운드 구조
- 총 **4라운드** 진행
- 각 라운드 시작 시 **점수판(Score Board)**이 그리드 위에 오버레이로 표시됨
- 점수판에는 보너스 칸(+점수)과 페널티 칸(-점수)이 표시됨
- 블록으로 보너스 칸을 덮으면 점수 획득, 라운드 종료 시 빈 칸은 페널티

### 점수 계산
| 항목 | 점수 |
|---|---|
| 보너스 칸 덮기 | +2점 |
| 일반 칸 채우기 | +1점 |
| 빈 칸 (라운드 종료 시) | -1점 |
| 보너스 칸 미달성 | -2점 |
| 그리드 완전 채우기 | +10점 보너스 |

## 테마 & 비주얼

### 세계관
현대 도시 개발 테마. 플레이어는 도시 설계사로서 블록 형태의 빌딩을 도시 부지에 배치한다.

### 색상 팔레트
- 배경: 어두운 네이비 `#1a1a2e`
- 그리드 빈 칸: 연한 회색 `#2d2d44`
- 그리드 선: `#3d3d5c`
- 블록 색상 (종류별 구분):
  - I: `#00b4d8` (하늘색)
  - O: `#f77f00` (주황)
  - T: `#9b5de5` (보라)
  - S: `#06d6a0` (민트)
  - Z: `#ef233c` (빨강)
  - L: `#ffd166` (노랑)
  - J: `#118ab2` (파랑)
- 보너스 칸: `#ffd700` (금색)
- 페널티 칸: `#ff4444` (빨강)

### UI 구성

### 애니메이션
- 블록 슬라이딩: 위→아래 트윈 (0.15초)
- 블록 배치 확정: 작은 펄스 효과
- 라운드 전환: 페이드 인/아웃
- 점수 획득: 숫자 팝업 (+2, -1 등)

## 게임 플로우
```
타이틀 화면
    ↓ [START]
라운드 1 시작 (점수판 공개)
    ↓ 카드 뽑기 → 열 선택 → 배치 (덱 소진까지 반복)
라운드 1 결산 (점수 계산 + 애니메이션)
    ↓
라운드 2~4 반복
    ↓
최종 결과 화면 (총점 + 등급)
    ↓ [PLAY AGAIN] / [SHARE]
```

## 등급 시스템
| 점수 | 등급 |
|---|---|
| 80점 이상 | S — 마스터 플래너 |
| 60~79점 | A — 수석 건축가 |
| 40~59점 | B — 도시 설계사 |
| 20~39점 | C — 견습 개발자 |
| 20점 미만 | D — 재개발 필요 |

## 타겟 유저
- itch.io 캐주얼 게임 유저
- 테트리스/퍼즐 게임 좋아하는 유저
- 한 판에 5~10분 가볍게 즐기고 싶은 유저

- 가격: **무료 (Pay What You Want)**
- 태그: puzzle, casual, tetris, solitaire, board-game, city-builder
- 언어: 영어 (게임 내), 한국어 (개발자 설명 추가)
- 커버 이미지: 630×500px
- 스크린샷: 3~5장 (각 라운드 + 결과화면)
